# Superseded: see MANUSCRIPT.md

This early technical note presented the core energy model. It has been **consolidated
into and superseded by [`MANUSCRIPT.md`](MANUSCRIPT.md)**, which is the single canonical
write-up of the model (same content, current numbers, with abstract, methods, validation,
sensitivity, figures, and references).

core papers in sync produced divergent numbers. The model's results live in one place now.
